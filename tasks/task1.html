Caesar Cipher

A caesar cipher wheel is an encryption tool that helps you hide messages! It has two wheels. You can spin the smaller wheel a specific number of places known as the key. This shifts the alphabet allowing you to make a hidden message by writing it using the corresponding letter.

Example: A sentence was encrypted with a shift of 7. The result of this is “FVB JYHJRLK AOL JVKL”. What was the original sentence?
Steps:
1) Write out the alphabet.
2) Next to the original alphabet, write out the shifted alphabet next to it.
3) Find each letter of the encrypted sentence in the shifted version of the alphabet.
4) Take the original letter of the alphabet it corresponds to.
5) Put these letters together to find the original sentence.

Answer = You cracked the code

Caesar Cipher Wheel

Encoding:
1) Find each letter of your message on the inner ring.
2) Move along the wheel until you find the corresponding shifted letter on the outer ring.
3) Write down the new letter from the outer ring.

Decoding:
To decode, you do the reverse process.
1) Align the inner and outer rings based on the shift you used for encoding.
2) Find the letter on the outer ring and move to the inner ring to find the original letter.


Vigenere

A vigenere table is a big grid with all the letters of the alphabet written across the top and down the side.

Example: The message is “LXFOPVEFRNHR”. Decrypt this using the key “Lemon”.
Steps:
1) Write out the key to fit the length of the message. (LEMONLEMONLE)
2) Take the first letter of the key and the message.
3) Find the key letter in the left column and follow the row until you reach your message letter. (In this case, follow the 'L' row until you find 'L'.)
4) Find the top row plaintext value this aligns with. (The top row shows the plaintext letter 'A').
5) Repeat for all letters in the message.
6) Put it together.

Answer = Attack at dawn

ASCII

ASCII is a system used to represent text and characters in computers. It assigns a unique number to each letter, number, or symbol you see on your screen. To use it, pick a letter, find it on the chart, and look at its matching number.
